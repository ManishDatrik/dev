# DEVBOD PCB Design

A PCB design project created using **KiCad** to learn the complete hardware design workflow, from schematic creation to PCB layout and manufacturing file generation.

---

## About

This project was designed in KiCad as part of my learning in PCB design. It includes creating the schematic, assigning footprints, designing the PCB layout, routing the connections, checking the design for errors, and generating the manufacturing files.

---
## Travel Purpose

Portable electronic devices have become an important part of travelling. A compact PCB like DEVBOD can serve as the foundation for travel-related embedded systems such as:

- Smart travel accessories
- Portable communication devices
- IoT-based travel gadgets
- Compact wireless hardware
- Outdoor and trekking electronics

This project demonstrates how PCB design can be applied to build small, portable hardware solutions for travel applications.

## Project Workflow

- Created the schematic
- Connected the required components
- Assigned footprints
- Designed the PCB layout
- Routed the PCB traces
- Performed Design Rule Check (DRC)
- Generated Gerber files for PCB manufacturing
